# TemuTriple 
## 여행 플랜 서비스 트리플 벤치마킹 서비스
- 해당 서비스는 혼자 여행하는 사람들이 손쉽게 여행 플랜을 생성할 수 있는 기능을 제공하는 서비스입니다.

## 개발환경
- opanai==1.61.1
- SQLAlchemy==0.0.22
- fastapi==0.115.8
- python==3.13.0

## 개발내용
- ChatGPT API키를 발급받아 챗봇 기능 구현
- jwt_token을 발급받아 해당 토큰을 쿠키에 담은 뒤에 프론트단에서 쿠키를 통해 사용자 검증
- 여행 플랜 계획 생성 및 삭제 기능 구현
